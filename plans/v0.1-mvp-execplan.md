# EncourageLoop v0.1 MVP ExecPlan

## Summary

Build the smallest useful EncourageLoop package: a deterministic Node.js CLI, a Codex skill, a plugin manifest, repo-scoped skill loading, local marketplace metadata, and blueprints/prompts that let Codex App continue work without re-planning the entire project each turn.

This plan is the execution contract. Older research and blueprints are boundary context, not the per-turn script.

## Execution Cursor

- Current baseline: new open-source repository at `https://github.com/YSCJRH/encourage-loop`
- Target: v0.1.0 MVP package
- Current stage: R0
- Stage status: in_progress
- Last completed evidence: initial scaffold pack generated
- Last validation: npm test should be run after unpacking
- Next atomic task: run `npm test`, inspect failures if any, and tighten the CLI behavior.
- Known blockers: none
- Scope guard:
  - no daemon
  - no keyboard injection
  - no LLM calls
  - no automatic merge/release
  - no abuse/whip branding
